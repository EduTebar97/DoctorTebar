# Modelo De Datos

Entidades principales:

- `User`: administradores y editores.
- `Post`: articulos de blog con categoria, tags, estado y SEO.
- `News`: noticias con fuente opcional.
- `Resource`: checklist, plantillas, guias, codigo y bibliografia.
- `Service`: servicios publicos editables.
- `Inquiry`: consultas recibidas desde contacto.
- `SiteSettings`: textos y configuracion publica.
- `AuditLog`: acciones relevantes de administracion.
- `MediaAsset`: medios subidos a Cloudinary.
