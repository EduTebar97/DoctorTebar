# Testing

## API

```bash
npm --workspace @doctor-tebar/api run test
```

Incluye pruebas de validacion, rutas protegidas y smoke tests de API.

## Frontend

```bash
npm --workspace @doctor-tebar/web run test
```

Incluye pruebas de componentes con Testing Library.

## E2E

```bash
npm --workspace @doctor-tebar/e2e run test
```

Requiere frontend y API en ejecucion.
