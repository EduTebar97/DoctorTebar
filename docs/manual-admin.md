# Manual Admin

## Entrar

1. Ir a `/login`.
2. Introducir email y password.
3. Acceder al panel `/admin`.

## Crear Articulo

1. Abrir `Blog`.
2. Pulsar `Nuevo`.
3. Rellenar titulo, extracto, categoria, tags y contenido.
4. Guardar como borrador o publicar.

## Consultas

1. Abrir `Consultas`.
2. Revisar el mensaje.
3. Cambiar estado a revisada, respondida o archivada.

## Ajustes

1. Abrir `Ajustes`.
2. Editar hero, sobre mi, email y color.
3. Guardar y verificar la parte publica.
